# Anime Website

A modern, dark-themed website for browsing and discovering anime.

## Features

- **Dark Theme:** A sleek and eye-catching dark mode design.
- **Home Page:**
  - **Header:** Navigation with Home, About, and Contact links.
  - **Logo:** Custom logo display.
  - **Greeting:** Welcoming text for visitors.
  - **Search Bar:** Real-time search to find anime or check availability.
  - **Category Filters:** Filter anime by genres like Action, Adventure, and Romance.
- **Scroll Reveal Animations:** Smooth fade-in and slide-up animations as you scroll through the page.
- **Details Page:**
  - **Comprehensive Info:** Name, release year, duration, episodes, genres, and language.
  - **Description:** A short summary of the anime.
  - **Download Buttons:** Easy access to download links for each episode.

## Technologies Used

- HTML5
- CSS3 (Custom Variables, Flexbox, Grid, Transitions)
- JavaScript (Vanilla, Intersection Observer API)

## How to View

Simply open `index.html` in any modern web browser.
